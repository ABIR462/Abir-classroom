Abacus

John von Neumann

Machine Language (Binary)

Device Driver

Spreadsheet Software (MS Excel)

Format Painter
8. Which option is used to replace one word with another?
Find and Replace
9. Name a box with dotted borders where text and pictures can be placed.
Placeholder
10. Which tab is used to change the background color of a slide?
Design Tab

Section B

1. Write any four characteristics of a computer.
Speed, Accuracy, Automation, Storage
2. What is Artificial Intelligence? Give two examples of AI used these days.
AI is the simulation of human intelligence by machines. Examples: Chatbots, Self-driving cars.
3. Write the full form of UNIVAC.
Universal Automatic Computer
4. Name two machines designed by Charles Babbage.
Difference Engine and Analytical Engine
5. Write a short note on Analytical Engine.
Designed by Charles Babbage, considered the first general-purpose computer concept. It had input, mill (processor), memory, and output.
6. Write any four functions of an operating system.
Memory management, Process management, File management, Device management
7. What is the use of a language translator? Name the different types.
It converts high-level language into machine language. Types: Compiler, Interpreter, Assembler.
8. Write a short note on Disk Defragmenter.
It rearranges fragmented data on a disk to improve access speed and efficiency.
9. What is Scanning? Why is it useful for a computer?
Scanning is the process of converting hard copy documents into digital format. Useful for storage and editing.
10. Differentiate between Find and Replace options.
Find: Locates text. Replace: Locates text and replaces it with new text.
11. Give one example each of Superscript and Subscript option. Name any four types of shapes which can be inserted in a document.
Superscript: x², Subscript: H₂O. Shapes: Rectangle, Circle, Arrow, Line.
12. How will you change the color scheme of a theme? Mention any three themes.
Go to Design tab → Themes → Colors. Examples: Office, Integral, Ion.
13. What is a Template? List any three templates.
A template is a ready-to-use design. Examples: Resume, Business Card, Invoice.
14. How can you change the background of a slide? Write the options available.
Design tab → Format Background. Options: Solid Fill, Gradient Fill, Picture/Texture Fill, Pattern Fill.
